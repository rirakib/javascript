#### জাভাস্ক্রিপ্ট কি?

জাভাস্ক্রিপ্ট হচ্ছে একটি স্ক্রিপ্টিং বা প্রোগ্রামিং লাঙ্গুয়েজ। বর্তমান সময়ে জাভাস্ক্রিপ্টকে বিভিন্ন উপায়ে ব্যবহার করা যাচ্ছে।
